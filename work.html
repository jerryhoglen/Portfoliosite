E-commerce Site Built Using Ruby on Rails

Technology used

  • HTML5
  • CSS3
  • JQuery
  • Ruby on Rails
  • AJAX
  • Heroku
  • RSpec
  • Payment forms with Stripe

This is an e-commerce website built using Ruby on Rails. This site has many features including the ability to create users with passwords and log in/log out as well as roles for admin. The page has reviews and ratings built in that allows the user to add reviews and ratings for products using AJAX for page refreshing without full reload. The back end was tested using Rspec and the page has full payment capabilities using the Stripe payment method.
